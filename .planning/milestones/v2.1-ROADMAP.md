# Milestone v2.1: UI Redesign

**Status:** SHIPPED 2026-01-31
**Phases:** 18-23
**Total Plans:** 15

## Overview

v2.1 modernizes the web interface with bento grid layouts and glassmorphism effects. Pure CSS approach using native backdrop-filter and CSS Grid provides sophisticated visual design without build tools or frameworks. Delivers professional scientific aesthetic with clear component separation while maintaining accessibility and mobile performance.

The roadmap follows a foundation-first strategy: establish design system and reusable components (Phase 18), then apply to each page in complexity order (Results most complex, Status simplest), wrap with responsive implementation, and validate with accessibility testing.

## Milestone Summary

**Key Decisions:**

- Pure CSS approach: No build tools, frameworks, or preprocessors needed
- Replace Pico CSS entirely: Custom CSS simpler than overriding framework for bento grids
- CSS Cascade Layers: Architectural organization without specificity wars
- BEM naming convention: Component boundaries prevent naming conflicts
- Glass opacity 85-95%: Higher opacity for WCAG contrast compliance
- Solid white background for viewer-card: WebGL/3Dmol.js has performance issues with glass/blur effects
- Touch-safe hover uses (pointer: fine) to prevent sticky hover on tap
- Shadow animation via pseudo-element opacity instead of box-shadow for 60fps performance

**Issues Resolved:**

- Pico CSS migration: Fully replaced with custom design system
- Safari backdrop-filter: Fixed with -webkit prefix using literal values
- Mobile performance: Reduced blur from 8-16px to 6-10px on mobile

**Issues Deferred:**

- Dark mode (DARK-01, DARK-02, DARK-03)
- Card expansion to focus mode (INTERACT-01)
- Drag-and-drop card reordering (INTERACT-02)
- User preference persistence (INTERACT-03)

**Technical Debt Incurred:**

- Minor breakpoint inconsistency (767px vs 768px) across files
- viewer-card defined in results-page.css rather than components/
