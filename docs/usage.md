# Usage Guide

> This guide will be completed in Phase 27.

## Topics to Be Covered

This guide will provide comprehensive usage documentation for:

- **Web UI Workflow**
  - Molecule input (SMILES, MOL, SDF)
  - Solvent and preset selection
  - Job submission and status tracking
  - Results interpretation

- **REST API Reference**
  - Authentication (if applicable)
  - Job submission endpoints with curl examples
  - Status polling and webhooks
  - Result retrieval endpoints
  - Error handling and status codes

- **Calculation Modes**
  - Single-conformer calculations
  - Ensemble mode with Boltzmann averaging
  - Draft vs production presets

- **Solvent Selection**
  - Available solvents (CHCl3, DMSO, gas phase)
  - COSMO solvation model details
  - When to use each solvent

- **Result Interpretation**
  - Chemical shift tables and units
  - Spectrum visualization features
  - 3D structure viewer with shift labels
  - Downloading results and raw data
