# Library Documentation

> This guide will be completed in Phase 29.

## Topics to Be Covered

This guide will document third-party library integrations:

- **RDKit**
  - SMILES parsing and validation
  - 3D conformer generation (ETKDG)
  - Molecule visualization and depiction
  - File format conversions

- **NWChem**
  - Input file generation
  - Output file parsing
  - Geometry optimization workflow
  - NMR shielding calculations

- **Huey**
  - Task queue configuration
  - Job scheduling and priorities
  - Consumer management
  - Result storage and retrieval

- **3Dmol.js**
  - Molecule viewer integration
  - Chemical shift label rendering
  - Interactive controls
  - Style customization

- **CREST/xTB (Optional)**
  - Conformer ensemble generation
  - GFN2-xTB force field
  - CREST metadynamics workflow
  - Integration with DFT pipeline

- **SmilesDrawer**
  - 2D structure preview
  - Real-time SMILES validation
  - Canvas rendering options
