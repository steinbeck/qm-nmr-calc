DELTA50 Molecule XYZ Files
==========================

This directory should contain 50 XYZ coordinate files named:
  molecule_01.xyz
  molecule_02.xyz
  ...
  molecule_50.xyz

Source: Supporting Information from Grimblat et al., Molecules 2023, 28(6), 2449
Download: https://www.mdpi.com/1420-3049/28/6/2449/s1

Due to download restrictions, these files must be obtained manually from the
publication's Supporting Information and placed in this directory.

Each XYZ file should follow standard XYZ format:
  Line 1: Number of atoms
  Line 2: Comment line (molecule name/ID)
  Lines 3+: Element X Y Z (one atom per line)
